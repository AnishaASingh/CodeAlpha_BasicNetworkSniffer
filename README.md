# Basic Network Sniffer

This project captures network packets using Python and Scapy.

## Features

- Captures live network traffic
- Displays Source IP Address
- Displays Destination IP Address
- Supports IPv4 and IPv6 packets

## Technologies Used

- Python
- Scapy
- Npcap

## How to Run

1. Install Python
2. Install Scapy:
   pip install scapy
3. Install Npcap
4. Run:
   python final_sniffer.py

## Output

The program displays:
- Source IP
- Destination IP
- Protocol information
